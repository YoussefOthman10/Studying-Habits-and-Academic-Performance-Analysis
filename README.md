# 📊 Studying Habits and Academic Performance Analysis

This project explores the relationship between university students’ study habits and their academic performance using survey data.

Through Exploratory Data Analysis (EDA) and data visualization techniques, the project examines how factors such as study hours, sleep duration, lecture attendance, and study methods relate to students’ Grade Point Average (GPA).

The analysis was conducted using Python and data science libraries to identify patterns and relationships within the dataset.

## 🎯 Project Objective

The main goal of this project is to analyze students’ study behaviors and investigate whether certain habits are associated with better academic outcomes.

The project focuses on answering the following research question:

How do studying habits such as study hours, sleep hours, and attendance relate to students' academic performance measured by GPA?

To explore this question, the project analyzes survey data collected from university students.

## 🗂 Dataset

The dataset was collected through an online Google Forms survey distributed among university students.

The survey gathered information about several variables related to studying habits and student lifestyle.

Variables included in the dataset
📚 Weekly study hours
🎓 GPA (Grade Point Average)
😴 Average sleep hours per day
🏫 Lecture attendance frequency
📅 Study frequency during the week
📝 When students start studying before exams
🧠 Preferred study methods
💼 Student employment status

Number of survey responses:
100 students

The dataset reflects real student behaviors but was collected using convenience sampling, which may limit generalization to all students.

## 🧪 Methodology

The analysis follows a standard Exploratory Data Analysis (EDA) workflow:

1️⃣ Data Collection

Survey responses were collected using Google Forms and exported as a dataset for analysis.

2️⃣ Data Cleaning

The dataset was inspected for potential errors or inconsistencies.

Detection of outliers using boxplots
Identification of an unrealistic GPA value
Correction of the data entry error to maintain dataset integrity
3️⃣ Exploratory Data Analysis

Several statistical and visualization techniques were used:

Distribution analysis
Outlier detection
Relationship analysis between variables
Correlation analysis
Behavioral pattern analysis

## 📊 Key Findings

The exploratory analysis revealed several insights:

Most students study between 5–15 hours per week
Most GPAs fall between 2.5 and 3.5
Students typically sleep 6–9 hours per night

However, correlation analysis shows that:

The relationship between study hours and GPA is weak
Sleep hours do not strongly correlate with GPA
Academic performance likely depends on multiple factors, not just study time

Additionally:

Many students begin studying a few days before exams
The most popular study method is solving problems or practice questions

## ⚠️ Limitations

Several limitations should be considered when interpreting the results:

The dataset was collected using convenience sampling
Survey responses were self-reported
The sample size (100 students) is relatively small
Results may not represent the entire student population

Despite these limitations, the dataset provides useful insight into student study habits.
